# Self-Custodial Wallet

Users create a self-custodial user wallet and a sensor account abstraction wallet via Particle Network by agreeing to the app's terms and conditions. The seed phrase is securely stored and never leaves the user's device. This sensor wallet provides Web3-based consent (powered by the Peaq Network) for the following:

*   Microphone and location access
*   Measurement transactions
*   Other user-related actions

Due to the app's reliance on specific phone capabilities, both the iOS and Android versions are developed natively, using Swift for iOS and Kotlin for Android.
