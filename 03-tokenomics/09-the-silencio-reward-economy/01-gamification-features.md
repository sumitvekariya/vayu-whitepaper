# Gamification Features

Making Data Collection Fun

Silencio transforms noise level measurement and smartphone-generated data collection into an engaging, gamified experience. These interactive elements motivate users to participate actively, turning data contribution into a fun and rewarding activity.

## Gamification Features

| Feature | Description |
| --- | --- |
| **Levels** | Users advance through levels by earning in-app Coins. Each level unlocks new perks and rewards, encouraging continuous participation. |
| **Badges** | Users earn badges for achieving specific milestones, such as measuring noise levels in a new city or inviting friends to the network. |
| **Leaderboards** | Users can compete with others on global and local leaderboards, fostering a sense of community and friendly competition. |
| **Missions** | Users can complete daily and weekly missions to earn extra in-app Coins and other rewards. |
| **Streaks** | Users are rewarded for consecutive days of activity, encouraging consistent engagement with the app. |
